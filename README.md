# RecallMax — God-Tier Long-Context Memory

## Installation
To install RecallMax, use the following command:

```bash
npx skills add christopherlhammer11-ai/recallmax
```

## Overview
RecallMax enhances AI agent memory capabilities by providing massive context injection, advanced summarization, and universal compatibility with leading-edge AI models like Gemini, Claude, GPT, and Grok.

## Key Features:
- **Massive Context Injection:** Easily injects 500k to 1M clean tokens without hallucination drift.
- **Advanced Summarization:** Auto-summarizes conversations with retention of tone, sarcasm, and intent.
- **Fact Verification:** Built-in mechanisms for quick fact-checks on controversial claims.
- **Efficient Compression:** Compresses multi-turn histories into high-density token sequences.
